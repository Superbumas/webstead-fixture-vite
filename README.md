# webstead-fixture-vite

Pinned test fixture for Webstead's "Deploy from GitHub" import path — the **Vite build** case.

- Root directory: `.`
- Output directory: `dist`
- Install: `npm install`
- Build: `npm run build`

Vite version is pinned so the build is deterministic. Used by `packages/icp-deploy/src/github-import.e2e.ts`.
